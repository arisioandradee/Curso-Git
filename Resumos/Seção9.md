# Markdown do básico ao avançado

## O que é Markdown?
É uma forma de adicionar estilo a textos na web, muito utilizado em arquivos README.md. Nele, você pode incluir trechos de códigos, links, imagens e muito mais.

## Criando títulos
Os cabeçalhos em Markdown são determinados pelo "#", sendo os famosos títulos ou headings do HTML. Utilize "# 1", "## 2", "### 3" para diferentes níveis.

## Ênfase
Podemos dar ênfase ao texto usando símbolos. Podemos escrever em **negrito** ou em _itálico_. Também é possível combinar os dois: _um **texto** combinado_.

## Listas
Markdown suporta listas ordenadas e não ordenadas. As listas não ordenadas começam os itens com: "* item", e as listas ordenadas com: "1. item".

## Imagens
Adicionamos imagens em Markdown usando a sintaxe: `![texto](imagem)`. A imagem pode ser externa ou estar no próprio repositório.

## Links
A sintaxe para links em Markdown é semelhante à de imagens: `[texto](url)`.

## Código
Para inserir códigos em Markdown, utilize três crases simples antes e depois do texto. Exemplo: \``` teste \```.

## Task List
Podemos criar uma lista de tarefas no Markdown. A sintaxe é: "[x] Texto" para concluídas e "[ ] Texto" para não concluídas.

Este é um Markdown que criei para explicar justamente o Markdown. No entanto, os textos entre aspas não devem adquirir o efeito de Markdown, pois é para o usuário visualizar a sintaxe. Também corrigi alguns erros de português.
